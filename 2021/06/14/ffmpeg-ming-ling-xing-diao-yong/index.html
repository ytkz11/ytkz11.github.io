Some ffmpeg skill

Posted on 2021-06-14 | In ffmpeg

Compress Size Of Vedio

ffmpeg -i input.mp4 -vcodec libx265 -crf 28 output.mp4

M4a Combine

ffmpeg -i "concat:first.m4a|second.m4a" -c copy out.m4a

FFmpeg 裁剪视频画面

ffmpeg -i 1.mp4 -vf crop=480:250:0:40 -threads 5 -preset ultrafast -strict -2 outputname.mp4

m4a combine

ffmpeg -i "concat:0.m4a|1.m4a" -safe 0 -c copy out.m4a

合并左右 , 并且压低画质

ffmpeg -i 9.mp4 -i 10.mp4 -filter_complex hstack -crf 50 output.mp4

压低画质

ffmpeg -i 9.mp4 -crf 50 -threads 1 output.mp4

m4a to aac

ffmpeg -i in.m4a -acodec copy out.aac

aac audio combine

ffmpeg -i "concat:0.aac|1.aac" -c copy result.aac

image to video

一秒一帧， 总时长18秒

ffmpeg -loop 1 -y -i %01d.jpg -vcodec libx264 -r 1 -t 18 test.mp4

剪切高度

ffmpeg -y -i 1.flv -vf scale=640:320 -thread 3 -acodec aac -vcodec h264 out.mp4

将视频中指定区域之外的部分进行模糊

ffmpeg -i a.mp4 -threads 1 -filter_complex "[0:v]boxblur=80[bg];[0:v]crop=1600:150:0:900[fg];[bg][fg]overlay=0:900" -map 0:v output.mp4

加logo

ffmpeg -i a.mp4 -i logo.png -filter_complex "[1:v] scale=400:204[logo];[0:v][logo]overlay=x=100:y=0" output.mp4

ffmpeg -i out.mp4 -i test.mp3 -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4

you-get

you-get --format=dash-flv480 url
you-get --format=dash-flv url
