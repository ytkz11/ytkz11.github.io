Windows安装Rust版本Gdal步骤 [

1、下载gdal c++编译后源码，将解压后的的文件放在同一个文件夹内

2、设置环境变量,注意gdal版本

// windows path系统环境变量
Path:C:\xxxxxx\gdal_lib\gdal\\bin
Path:C:\xxxxxx\gdal_lib\gdal\\bin\\gdal\\apps

// windows 全局系统环境变量
GDAL_HOME=C:\xxxxxx\gdal_lib\gdal
PKG_CONFIG_PATH=C:\xxxxxx\gdal_lib\gdal
PROJ_LIB=C:\xxxxxx\gdal_lib\gdal\bin\proj9\share
GDAL_VERSION=373

3、Chocolatey、pkgconfiglite安装

下载地址：https://github.com/chocolatey/choco/releases

安装chocolatey后，在命令行输出以下语句，安装pkgconfiglite


choco install pkgconfiglite

4、查看gdal版本

gdalinfo --version
//GDAL 3.7.3, released 2023/10/30

5、写入gdal.pc文件,注意**Version:3.7.3要和第四步相同

gdal.pc文件路径 C:\xxxxxx\gdal_lib\gdal\gdal.pc

gdal.pc文件内容

name=gdal
prefix=/usr
exec_prefix=${prefix}
libdir=${exec_prefix}/lib
includedir=${exec_prefix}/include
datadir=${prefix}/share/${name}

Name: lib${name}
Description: Geospatial Data Abstraction Library
Version: 3.7.3
Libs: -L${libdir} -l${name}
Cflags: -I${includedir}/${name}

5、运行官方示例

https://github.com/georust/gdal

use gdal::Dataset;

fn main() {
    // Open a raster dataset
    let dataset = Dataset::open("D:\\GF2_PMS1_E102.3_N38.1.tif").unwrap();

    // Access raster properties
    let width = dataset.raster_size().0;
    let height = dataset.raster_size().1;
    let band_count = dataset.raster_count();
        
    println!("Raster width: {}", width);
    println!("Raster height: {}", height);
    println!("Band count: {}", band_count);
}

成功把rust-gdal在win10下进行调用。
