安装gin

gin是go语言在构建web的框架。

在项目的目录命令行cmd运行：

go get -u github.com/gin-gonic/gin

在本地安装gin。

package main

import "github.com/gin-gonic/gin"

func main() {
	r := gin.Default()
	r.GET("/lala", func(c *gin.Context) {
		c.JSON(200, gin.H{
			"message": "hello",
		})
	})
	r.Run() //  127.0.0.1:8080
}

package main表示一个可独立执行的程序，每个Go 应用程序都包含一个名为main 的包。

import “github.com/gin-gonic/gin” 导入gin包。

func main() Go 语言程序有一个特殊的函数，就是 main 函数，main 函数是程序的入口，也就是说程序的运行一定是从 main 函数开始的。
