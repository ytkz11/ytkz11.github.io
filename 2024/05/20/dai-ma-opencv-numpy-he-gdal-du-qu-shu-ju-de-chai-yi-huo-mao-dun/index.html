OpenCVгҖҒNumPyе’ҢGDALиҜ»еҸ–ж•°жҚ®зҡ„е·®ејӮжҲ–зҹӣзӣҫ

ytkz2024-05-202024-05-20

еңЁеӣҫеғҸеӨ„зҗҶе’Ңең°зҗҶз©әй—ҙж•°жҚ®еӨ„зҗҶдёӯпјҢOpenCVгҖҒNumPyе’ҢGDALжҳҜеёёз”Ёзҡ„еә“гҖӮз„¶иҖҢпјҢе®ғд»¬еңЁж•°жҚ®иҜ»еҸ–е’Ңеқҗж Үеј•з”Ёж–№ејҸдёҠеӯҳеңЁе·®ејӮпјҢеҸҜиғҪеҜјиҮҙдҪҝз”Ёж—¶еҮәзҺ°зҹӣзӣҫгҖӮд»ҘдёӢжҳҜеҜ№иҝҷдёүиҖ…иҜ»еҸ–ж•°жҚ®зҡ„иЎҢеҲ—е·®ејӮеҸҠзҹӣзӣҫзҡ„еҲҶжһҗпјҡ

OpenCV

  1. ж•°жҚ®иҜ»еҸ–ж–№ејҸпјҡ
    • OpenCVдҪҝз”Ёcv2.imread()ж–№жі•иҜ»еҸ–еӣҫеғҸпјҢиҝ”еӣһдёҖдёӘNumPyж•°з»„гҖӮ
  2. ж•°жҚ®ж јејҸпјҡ
    • иҜ»еҸ–зҡ„еӣҫеғҸж•°жҚ®ж јејҸдёә(height, width, channels)пјҢеҚі(иЎҢж•°, еҲ—ж•°, йҖҡйҒ“ж•°)гҖӮ
  3. еқҗж Үеј•з”Ёж–№ејҸпјҡ
    • еңЁOpenCVдёӯпјҢеқҗж Үзі»з»ҹйҒөеҫӘеӣҫеғҸеӨ„зҗҶзҡ„жғҜдҫӢпјҢ(i, j)иЎЁзӨәз¬¬iиЎҢгҖҒз¬¬jеҲ—зҡ„еғҸзҙ гҖӮ

NumPy

  1. ж•°жҚ®иҜ»еҸ–ж–№ејҸпјҡ
    • NumPyеҸҜд»ҘйҖҡиҝҮеӨҡз§Қж–№ејҸиҜ»еҸ–ж•°жҚ®пјҢдҫӢеҰӮnumpy.loadtxt()гҖҒnumpy.genfromtxt()зӯүгҖӮ
  2. ж•°жҚ®ж јејҸпјҡ
    • NumPyж•°з»„зҡ„й»ҳи®Өж јејҸдёә(rows, cols)пјҢеҚі(иЎҢж•°, еҲ—ж•°)гҖӮеҜ№дәҺеӨҡз»ҙж•°з»„пјҢж јејҸдёә(depth, rows, cols)гҖӮ
  3. еқҗж Үеј•з”Ёж–№ејҸпјҡ
    • NumPyж•°з»„дёӯзҡ„е…ғзҙ йҖҡиҝҮ(i, j)и®ҝй—®пјҢеҚіз¬¬iиЎҢгҖҒз¬¬jеҲ—гҖӮ

GDAL (Geospatial Data Abstraction Library)

  1. ж•°жҚ®иҜ»еҸ–ж–№ејҸпјҡ
    • GDALз”ЁдәҺиҜ»еҸ–е’ҢеӨ„зҗҶең°зҗҶз©әй—ҙж•°жҚ®пјҢеҰӮйҒҘж„ҹеӣҫеғҸе’ҢGISж•°жҚ®гҖӮеёёз”Ёж–№жі•жҳҜgdal.Open()е’ҢReadAsArray()гҖӮ
  2. ж•°жҚ®ж јејҸпјҡ
    • GDALиҜ»еҸ–зҡ„ж•°жҚ®ж јејҸдёә(bands, rows, cols)пјҢеҚі(жіўж®өж•°, иЎҢж•°, еҲ—ж•°)гҖӮ
  3. еқҗж Үеј•з”Ёж–№ејҸпјҡ
    • еңЁGDALдёӯпјҢеқҗж Үеј•з”Ёзі»з»ҹдёҺең°зҗҶеқҗж Үзі»з»ҹзӣёеҜ№еә”пјҢrowsйҖҡеёёеҜ№еә”дәҺyеқҗж ҮпјҲзә¬еәҰпјүпјҢcolsеҜ№еә”дәҺxеқҗж ҮпјҲз»ҸеәҰпјүгҖӮ

иЎҢеҲ—е·®ејӮдёҺзҹӣзӣҫеҲҶжһҗ

  1. еқҗж Үзі»з»ҹе’Ңж•°жҚ®ж јејҸзҡ„е·®ејӮпјҡ
    • OpenCVдёҺNumPyпјҡеңЁеҚ•йҖҡйҒ“жғ…еҶөдёӢпјҢOpenCVдёҺNumPyзҡ„еқҗж Үеј•з”Ёж–№ејҸе’Ңж•°жҚ®ж јејҸдёҖиҮҙпјҢйғҪжҳҜ(rows, cols)гҖӮеңЁеӨҡйҖҡйҒ“жғ…еҶөдёӢпјҢOpenCVзҡ„ж•°жҚ®ж јејҸдёә(rows, cols, channels)пјҢиҖҢNumPyжІЎжңүзү№е®ҡзҡ„еӨҡйҖҡйҒ“ж јејҸгҖӮ
    • GDALпјҡGDALзҡ„(bands, rows, cols)ж јејҸдёҺOpenCVе’ҢNumPyзҡ„ж јејҸеӯҳеңЁз»ҙеәҰйЎәеәҸдёҠзҡ„е·®ејӮпјҢзү№еҲ«жҳҜеңЁеӨ„зҗҶеӨҡжіўж®өең°зҗҶз©әй—ҙж•°жҚ®ж—¶пјҢиҝҷз§Қе·®ејӮйңҖиҰҒзү№еҲ«жіЁж„ҸгҖӮ
  2. ең°зҗҶеқҗж ҮдёҺеӣҫеғҸеқҗж Үзҡ„жҳ е°„пјҡ
    • еңЁеӨ„зҗҶең°зҗҶз©әй—ҙж•°жҚ®ж—¶пјҢGDALзҡ„rowsе’ҢcolsдёҺең°зҗҶеқҗж Үзі»з»ҹдёӯзҡ„yпјҲзә¬еәҰпјүе’ҢxпјҲз»ҸеәҰпјүзӣёеҜ№еә”пјҢиҖҢOpenCVе’ҢNumPyзӣҙжҺҘдҪҝз”ЁеӣҫеғҸеқҗж Үзі»з»ҹгҖӮеңЁд»ҺGDALиҜ»еҸ–ж•°жҚ®е№¶дёҺOpenCVжҲ–NumPyз»“еҗҲдҪҝз”Ёж—¶пјҢйңҖиҰҒиҖғиҷ‘иҝҷз§Қең°зҗҶеқҗж Үе’ҢеӣҫеғҸеқҗж Үзҡ„жҳ е°„е…ізі»гҖӮ

и§ЈеҶіиЎҢеҲ—зҹӣзӣҫзҡ„ж–№жі•

  1. ж•°жҚ®иҪ¬жҚўпјҡ

    • еңЁдёҚеҗҢеә“д№Ӣй—ҙдәӨжҚўж•°жҚ®ж—¶пјҢйңҖиҰҒи°ғж•ҙж•°з»„зҡ„еҪўзҠ¶д»ҘзЎ®дҝқж•°жҚ®ж јејҸзҡ„дёҖиҮҙжҖ§гҖӮдҫӢеҰӮпјҢдҪҝз”Ёnp.transpose()и°ғж•ҙGDALж•°жҚ®зҡ„з»ҙеәҰйЎәеәҸгҖӮ
    pythonеӨҚеҲ¶д»Јз Ғimport numpy as np
from osgeo import gdal

# дҪҝз”ЁGDALиҜ»еҸ–ең°зҗҶз©әй—ҙж•°жҚ®
dataset = gdal.Open('image.tif')
array_gdal = dataset.ReadAsArray()
print('GDAL Array Shape:', array_gdal.shape)  # (bands, rows, cols)

# е°ҶGDALж•°з»„иҪ¬жҚўдёәNumPyж•°з»„ж јејҸ
array_gdal_transposed = np.transpose(array_gdal, (1, 2, 0))
print('Transposed GDAL Array Shape:', array_gdal_transposed.shape)  # (rows, cols, bands)

  2. еқҗж Үзі»з»ҹиҪ¬жҚўпјҡ

    • еҪ“еңЁең°зҗҶз©әй—ҙж•°жҚ®еӨ„зҗҶе’ҢеӣҫеғҸеӨ„зҗҶд№Ӣй—ҙиҝӣиЎҢиҪ¬жҚўж—¶пјҢйңҖиҰҒжіЁж„Ҹең°зҗҶеқҗж ҮдёҺеӣҫеғҸеқҗж Үзҡ„е…ізі»пјҢзЎ®дҝқж•°жҚ®еңЁиҪ¬жҚўиҝҮзЁӢдёӯдҝқжҢҒжӯЈзЎ®зҡ„ең°зҗҶдҝЎжҒҜгҖӮ

  3. дёҖиҮҙжҖ§жЈҖжҹҘпјҡ

    • еңЁиҝӣиЎҢж•°жҚ®еӨ„зҗҶеүҚеҗҺпјҢзЎ®дҝқйҖҡиҝҮжЈҖжҹҘе’Ңи°ғж•ҙж•°з»„зҡ„еҪўзҠ¶жқҘдҝқжҢҒж•°жҚ®ж јејҸзҡ„дёҖиҮҙжҖ§гҖӮдҪҝз”Ёжё…жҷ°зҡ„ж–ҮжЎЈе’ҢжіЁйҮҠиҜҙжҳҺж•°жҚ®ж јејҸе’ҢиҪ¬жҚўж–№жі•д№ҹжңүеҠ©дәҺеҮҸе°‘й”ҷиҜҜгҖӮ

д»Јз ҒзӨәдҫӢ

д»ҘдёӢзӨәдҫӢеұ•зӨәдәҶеҰӮдҪ•еңЁOpenCVгҖҒNumPyе’ҢGDALд№Ӣй—ҙиҝӣиЎҢж•°жҚ®иҪ¬жҚўе’Ңеқҗж ҮеӨ„зҗҶпјҡ

pythonеӨҚеҲ¶д»Јз Ғimport cv2
import numpy as np
from osgeo import gdal

# дҪҝз”ЁOpenCVиҜ»еҸ–еӣҫеғҸ
image_cv = cv2.imread('image.jpg')
print('OpenCV Image Shape:', image_cv.shape)  # (height, width, channels)

# е°ҶOpenCVеӣҫеғҸиҪ¬жҚўдёәNumPyж•°з»„пјҲе·ІжҳҜNumPyж•°з»„пјҢдҪҶеҪўзҠ¶дёҖиҮҙпјү
image_np = np.array(image_cv)
print('NumPy Image Shape:', image_np.shape)  # (height, width, channels)

# дҪҝз”ЁGDALиҜ»еҸ–ең°зҗҶз©әй—ҙж•°жҚ®
dataset = gdal.Open('image.tif')
array_gdal = dataset.ReadAsArray()
print('GDAL Array Shape:', array_gdal.shape)  # (bands, rows, cols)

# е°ҶGDALж•°з»„иҪ¬жҚўдёәNumPyж•°з»„ж јејҸ
array_gdal_transposed = np.transpose(array_gdal, (1, 2, 0))
print('Transposed GDAL Array Shape:', array_gdal_transposed.shape)  # (rows, cols, bands)

# зЎ®дҝқдёҖиҮҙжҖ§
assert image_np.shape[0:2] == array_gdal_transposed.shape[0:2], "Shape mismatch!"

# еқҗж Үзі»з»ҹиҪ¬жҚўзӨәдҫӢпјҡд»Һең°зҗҶеқҗж ҮеҲ°еӣҫеғҸеқҗж Ү
# еҒҮи®ҫжҲ‘д»¬жңүдёҖдёӘең°зҗҶеқҗж Ү (latitude, longitude)
latitude = 40.7128
longitude = -74.0060

# иҺ·еҸ–ең°зҗҶеқҗж ҮиҪ¬жҚўеҸӮж•°  
geotransform = dataset.GetGeoTransform()  
  
# е°Ҷең°зҗҶеқҗж ҮиҪ¬жҚўдёәеӣҫеғҸеқҗж ҮпјҲжіЁж„ҸиҝҷйҮҢжҲ‘д»¬зӣҙжҺҘдҪҝз”ЁдәҶgeotransformпјү  
pixel_x = int((longitude - geotransform[0]) / geotransform[1])  
pixel_y = int((geotransform[3] - latitude) / abs(geotransform[5]))  # жіЁж„ҸиҝҷйҮҢдҪҝз”ЁдәҶabsжқҘзЎ®дҝқеҲҶжҜҚдёәжӯЈ  
  
print(f'Geographic Coordinates ({latitude}, {longitude}) -> Image Coordinates ({pixel_x}, {pixel_y})')
