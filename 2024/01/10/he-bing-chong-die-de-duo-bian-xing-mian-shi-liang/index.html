дј з»ҹж“ҚдҪң

йқўзҹўйҮҸеҗҲе№¶пјҢеңЁarcgisжҳҜдёҖдёӘеёёи§Ғзҡ„ж“ҚдҪңгҖӮдҪҝз”ЁArcGISеҗҲе№¶йҮҚеҸ зҡ„еӨҡиҫ№еҪўзҡ„жӯҘйӘӨеҰӮдёӢпјҡ

  1. еҮҶеӨҮж•°жҚ®пјҡйҰ–е…ҲйңҖиҰҒеҮҶеӨҮеҫ…еӨ„зҗҶзҡ„йҮҚеҸ еӨҡиҫ№еҪўж•°жҚ®пјҢиҝҷеҸҜд»ҘйҖҡиҝҮйҒҘж„ҹеҪұеғҸи§ЈиҜ‘гҖҒйҮҺеӨ–е®һең°и°ғжҹҘзӯүж–№ејҸиҺ·еҫ—гҖӮ
  2. жһ„е»әйҮҚеҸ еҢәеҹҹеӣҫеұӮпјҡдҪҝз”ЁArcGISзҡ„вҖқUnionвҖқе·Ҙе…·пјҢе°ҶйҮҚеҸ зҡ„еӨҡиҫ№еҪўж•°жҚ®еұӮеҗҲе№¶дёәдёҖдёӘж–°еӣҫеұӮгҖӮ
  3. иҝӣиЎҢжӢ“жү‘еҗҲе№¶пјҡдҪҝз”ЁArcGISзҡ„вҖқTopologicalUnionвҖқе·Ҙе…·пјҢеҜ№йҮҚеҸ еҢәеҹҹеӣҫеұӮиҝӣиЎҢжӢ“жү‘еҗҲе№¶гҖӮиҝҷдёӘиҝҮзЁӢдјҡиҮӘеҠЁе°ҶйҮҚеҸ еҢәеҹҹеҗҲе№¶дёәдёҖдёӘж–°зҡ„еӨҡиҫ№еҪўгҖӮ

е®ҢжҲҗдёҠиҝ°жӯҘйӘӨеҗҺпјҢеҚіеҸҜе®ҢжҲҗеӨҡиҫ№еҪўзҡ„еҗҲе№¶гҖӮжіЁж„ҸпјҢд»ҘдёҠжӯҘйӘӨеҸҜиғҪдјҡж №жҚ®е…·дҪ“зҡ„йҮҚеҸ жғ…еҶөе’Ңж•°жҚ®зұ»еһӢжңүжүҖеҸҳеҢ–пјҢйңҖиҰҒе…·дҪ“й—®йўҳе…·дҪ“еҲҶжһҗгҖӮ

йӮЈеҰӮдҪ•з”Ёд»Јз Ғе®һзҺ°иҝҷдёӘеҠҹиғҪе‘ўпјҹ

жөӢиҜ•ж•°жҚ®

д»Ҙеҫ·еӣҪзҹўйҮҸдҪңдёәжөӢиҜ•ж•°жҚ®гҖӮ

д»Јз Ғ

#!/usr/bin/env python
# -*- coding: utf-8 -*- 
# @File : DissolveOverlappingPolygons.py
# еҗҲе№¶йҮҚеҸ зҡ„еӨҡиҫ№еҪў

import fiona
from shapely.ops import unary_union
from shapely.geometry import shape, mapping

def dissolve_overlapping_polygons(src, dst):
    '''
    Dissolve overlapping polygons in a shapefile
    @param src: input shapefile
    @param dst: output shapefile
    @return:
    '''
    with fiona.open(src, 'r') as ds_in:
        crs = ds_in.crs
        drv = ds_in.driver

        filtered = filter(lambda x: shape(x["geometry"]).is_valid, list(ds_in))

        geoms = [shape(x["geometry"]) for x in filtered]
        dissolved = unary_union(geoms)

    schema = {
        "geometry": "Polygon",
        "properties": {"id": "int"}
    }

    with fiona.open(dst, 'w', driver=drv, schema=schema, crs=crs) as ds_dst:
        try:
            for i,g in enumerate(dissolved.geoms):
                ds_dst.write({"geometry": mapping(g), "properties": {"id": i}})
        except Exception as e:
            print(e)

д»Јз ҒеҲҶжһҗ

иҝҷж®өд»Јз Ғзҡ„зӣ®зҡ„жҳҜеҗҲе№¶йҮҚеҸ зҡ„еӨҡиҫ№еҪўгҖӮд»ҘдёӢжҳҜд»Јз Ғзҡ„иҜҰз»ҶеҲҶжһҗпјҡ

  1. еҜје…Ҙеҝ…иҰҒзҡ„еә“пјҡ
    • fionaпјҡз”ЁдәҺиҜ»еҶҷең°зҗҶз©әй—ҙж•°жҚ®ж јејҸпјҢеҰӮShapefileгҖӮ
    • shapely.ops.unary_unionпјҡз”ЁдәҺе°ҶеӨҡдёӘеҮ дҪ•еҜ№иұЎеҗҲе№¶дёәдёҖдёӘеҜ№иұЎгҖӮ
    • shapely.geometryпјҡз”ЁдәҺеҲӣе»әгҖҒзј–иҫ‘е’ҢеҲҶжһҗеҮ дҪ•еҜ№иұЎгҖӮ
  2. е®ҡд№үdissolve_overlapping_polygonsеҮҪж•°пјҡ
    • еҸӮж•°
      • srcпјҡиҫ“е…Ҙзҡ„Shapefileи·Ҝеҫ„гҖӮ
      • dstпјҡиҫ“еҮәзҡ„Shapefileи·Ҝеҫ„гҖӮ
    • еҠҹиғҪпјҡиҜ»еҸ–иҫ“е…Ҙзҡ„ShapefileпјҢеҗҲе№¶йҮҚеҸ зҡ„еӨҡиҫ№еҪўпјҢе№¶е°Ҷз»“жһңеҶҷе…ҘеҲ°иҫ“еҮәж–Үд»¶гҖӮ
  3. еӨ„зҗҶShapefileпјҡ
    • дҪҝз”Ёfiona.openжү“ејҖиҫ“е…Ҙзҡ„ShapefileпјҢе№¶иҺ·еҸ–е…¶еқҗж ҮеҸӮиҖғзі»з»ҹпјҲCRSпјүе’Ңй©ұеҠЁзЁӢеәҸгҖӮ
    • дҪҝз”ЁfilterеҮҪж•°е’Ңis_validж–№жі•зӯӣйҖүеҮәжңүж•Ҳзҡ„еҮ дҪ•еҜ№иұЎпјҲеҚійқһиҮӘдәӨзҡ„еӨҡиҫ№еҪўпјүгҖӮ
    • е°ҶзӯӣйҖүеҗҺзҡ„еҮ дҪ•еҜ№иұЎиҪ¬жҚўдёәShapelyеҜ№иұЎпјҢе№¶дҪҝз”Ёunary_unionеҮҪж•°иҝӣиЎҢеҗҲе№¶гҖӮ
  4. е®ҡд№үиҫ“еҮәж–Үд»¶зҡ„жЁЎејҸпјҡ
    • е®ҡд№үдәҶиҫ“еҮәзҡ„Shapefileзҡ„жЁЎејҸпјҢе…¶дёӯеҮ дҪ•зұ»еһӢдёәвҖқPolygonвҖқпјҢеұһжҖ§дёӯжңүдёҖдёӘвҖқidвҖқеӯ—ж®өгҖӮ
  5. еҶҷе…Ҙиҫ“еҮәж–Үд»¶пјҡ
    • дҪҝз”Ёfiona.openжү“ејҖиҫ“еҮәж–Үд»¶пјҢе№¶дҪҝз”Ёе®ҡд№үзҡ„schemaе’ҢCRSгҖӮ
    • йҒҚеҺҶеҗҲе№¶еҗҺзҡ„еӨҡиҫ№еҪўпјҢе°ҶжҜҸдёӘеӨҡиҫ№еҪўзҡ„еҮ дҪ•дҝЎжҒҜе’ҢеұһжҖ§пјҲеңЁиҝҷйҮҢпјҢеұһжҖ§еҸӘжңүдёҖдёӘidпјүеҶҷе…ҘеҲ°иҫ“еҮәж–Үд»¶гҖӮ
  6. дё»еҮҪж•°пјҡе®ҡд№үдәҶиҫ“е…Ҙе’Ңиҫ“еҮәж–Үд»¶зҡ„и·Ҝеҫ„пјҢе№¶и°ғз”ЁдёҠиҝ°еҮҪж•°иҝӣиЎҢеҗҲе№¶ж“ҚдҪңгҖӮ
  7. ејӮеёёеӨ„зҗҶпјҡеҰӮжһңеңЁеҶҷе…ҘиҝҮзЁӢдёӯеҮәзҺ°ејӮеёёпјҢдјҡжҚ•иҺ·иҜҘејӮеёёе№¶жү“еҚ°еҮәжқҘгҖӮ

жіЁж„Ҹпјҡд»Јз ҒдёӯжІЎжңүеӨ„зҗҶдёҚеҗҢеқҗж ҮеҸӮиҖғзі»з»ҹзҡ„й—®йўҳпјҢеҰӮжһңиҫ“е…Ҙе’Ңиҫ“еҮәж–Үд»¶зҡ„CRSдёҚеҗҢпјҢеҸҜиғҪйңҖиҰҒиҝӣиЎҢеқҗж ҮиҪ¬жҚўгҖӮжӯӨеӨ–пјҢдёәдәҶд»Јз Ғзҡ„жӯЈеёёиҝҗиЎҢпјҢдҪ йңҖиҰҒзЎ®дҝқе®үиЈ…дәҶжүҖжңүеҝ…иҰҒзҡ„еә“пјҢеҢ…жӢ¬Fionaе’ҢShapelyгҖӮ
